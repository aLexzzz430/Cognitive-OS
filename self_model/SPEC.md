# Self Model — 自我能力、误差、可靠性建模

**唯一职责**：建模"我自己的能力边界"——什么能做、什么做不好、什么时候该信自己。

## 契约

**输入**：历史执行记录 + 当前资源状态
**输出**：自我评估报告（能力置信度 / 误差估计 / 可靠性评分）
**可读状态**：state/, memory/
**禁止副作用**：不得直接修改 state/

## 核心能力

1. **能力评估**：对每个候选动作给出成功概率估计
2. **误差建模**：识别系统性的决策偏差
3. **可靠性评分**：当前状态下各模块输出的可信度
4. **元认知**：知道"自己知道什么不知道什么"

## 关键原则（来自 V1）

- V1 的自我反思停留在日志层面 → V2 自我模型必须参与决策
- 自我模型输出应影响 policy 的候选评分
- 自我模型的"不知道"本身是需要决策的输入

## 约束

- 自我模型不准直接拒绝动作（只能调低评分，由 governance 最终决定）
- 自我模型不准硬编码"我已经足够好了"这类元认知断言
- 可靠性评分必须可审计（记录依据）
